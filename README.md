# md2wechat 📱

**Markdown → WeChat Official Account Article Converter**

将Markdown格式的文章一键转换为微信公众号文章样式HTML。

## ✨ Features

- ✅ Code syntax highlighting
- ✅ WeChat-compatible CSS inline styles
- ✅ Table support
- ✅ Image placeholder
- ✅ Blockquote style
- ✅ Copy to clipboard
- ✅ Preview in browser
- ✅ Pure Python, no dependencies!

## 🚀 Quick Start

```bash
# Convert markdown to HTML
python md2wechat.py article.md -o article.html

# Preview in browser
python md2wechat.py article.md --preview

# Copy to clipboard
python md2wechat.py article.md --copy
```

## 📦 Installation

```bash
git clone https://github.com/chengjunhua465-create/md2wechat.git
cd md2wechat
```

Requirements: Python 3.6+

## 📝 Example

Input `article.md`:
```markdown
# Hello World

This is **bold** text and *italic* text.

- Item 1
- Item 2

> This is a quote

[Link](https://example.com)
```

Output: Beautiful WeChat-compatible HTML!
